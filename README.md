# HR-analytics-Dash-Board
HR Analytics Dashboard is a data analysis project focused on understanding employee attrition and workforce trends using HR data.
The project involves cleaning and transforming raw HR data, analyzing key metrics such as employee count, attrition rate, average age, job roles, and departments, and presenting insights through an interactive dashboard.
Using Power BI, the dashboard helps HR teams identify patterns behind employee attrition and supports data-driven decision-making to improve employee retention and workforce planning.
